# AgroLegHum v1.0

## Hardware:

| Hw                                                                                            | Tipo           |
| ----                                                                                          | -----          |
| Arduino nano                                                                                  | Motherboard    |
| [RTC DS1307](https://naylampmechatronics.com/blog/52_tutorial-rtc-ds1307-y-eeprom-at24c.html) | RTC            |
| LCD 1602                                                                                      | Pantalla       |
| [AHT10](https://learn.adafruit.com/adafruit-aht20/arduino)                                    | Sensor Humedad |
